# Log!t

**Version:** v1.0.5
**Created by:** [Suz41](https://github.com/Suz41)
**Live:** https://suz41.github.io/Logit

Personal movie logger. Search via TMDB, rate, and track — all stored locally.

## Pages

- **index.html** — main app (search, rate, library)
- **PS.html** — stats dashboard
- **about.html** — tech stack & storage info

## Tech

HTML / CSS / JS (single file) | TMDB API | localStorage | GitHub Pages
